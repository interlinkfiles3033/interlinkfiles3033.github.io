5.8 Permutations and Combinations

Big Picture

Permutations and combinations are methods used to count the number of ways we can arrange or select objects from a set. The key difference between the two is that in permutations, the order of the objects matters, while in combinations, the order does not matter. These concepts are essential in probability and statistics for determining possible outcomes and arrangements.

Key Terms

  • Probability: The chance that an event will happen, written as a fraction, decimal, or percent.
  • Event: A set of one or more possible outcomes in a probability experiment.
  • Outcomes: Possible end results of an experiment.
  • Sample Space: The set of all possible outcomes of an experiment.
  • Fundamental Counting Principle: If an event can be chosen in p ways and another independent event in q ways, the total number of arrangements is p × q.
  • Independent Events: Events where the occurrence of one does not impact the occurrence of the other.
  • Permutation: An arrangement of objects where the order is important.
  • Factorial (n!): The product of all positive integers up to n (e.g., 5! = 5 × 4 × 3 × 2 × 1).
  • Combination: A selection of items where the order does not matter.

Important Formulas

- **Permutation Formula**: The number of ways to arrange r objects from a set of n objects is:

P(n, r) = n! / (n - r)!

Where:

  • n is the total number of objects,
  • r is the number of objects to be arranged,
  • n! is the factorial of n.

- **Combination Formula**: The number of ways to choose r objects from a set of n objects is:

C(n, r) = n! / [r! (n - r)!]

Where:

  • n is the total number of objects,
  • r is the number of objects to be selected,
  • r! is the factorial of r, and
  • n! is the factorial of n.

Example: Arranging Books on a Shelf

Suppose you have 5 different books and you want to arrange 3 of them on a shelf. Since the order of the books matters in this case, we use the permutation formula.

Using the permutation formula: P(n, r) = n! / (n - r)!

In this example, n = 5 (total number of books), and r = 3 (the number of books to arrange).

P(5, 3) = 5! / (5 - 3)! = 5! / 2! = (5 × 4 × 3 × 2 × 1) / (2 × 1) = 60

So, there are 60 different ways to arrange 3 books out of 5.

Conclusion

Permutations and combinations are fundamental techniques used in probability and statistics to count arrangements and selections of objects. By understanding these methods, you can determine how many possible outcomes or arrangements exist in various real-world situations, such as arranging items, selecting teams, or organizing events.
