4.3 Harmonic Mean

Summary Statement of Topic

The harmonic mean helps find the middle value of a data set regardless of how skewed the set might be by a few large values. This is achieved by using either weighted or unweighted averages, depending on whether the values are of equal importance or not.

Key Terms

  • Harmonic Mean: Calculated by dividing the number of values in the set by the sum of the inverses of the values in the set.
  • Arithmetic Mean: Also called the average; Arithmetic Mean = x₁ + x₂ + ... + xn.
  • Skewed/Skew: A given set means to cause the trend of data to favor one end or the other.
  • Weighted: A value or set of values takes into account varying levels of importance among members of the set.
  • Geometric Mean: A method of finding the “middle” value in a set that contains some values that are intrinsically more influential than others.
  • Data Set: A collection of these observations of the variable.
  • Weighted Harmonic Mean: A harmonic mean of values with varying frequencies or weights.

Example (Unweighted Harmonic Mean)

Basic/Unweighted Harmonic Mean Calculation:
Segment #1: 18 mph, 4.2 mi
Segment #2: 21 mph, 4.2 mi
Segment #3: 24 mph, 4.2 mi
Segment #4: 17 mph, 4.2 mi

To calculate the harmonic mean, we use the following formula:
Harmonic Mean = n / (1/x₁ + 1/x₂ + 1/x₃ + ... + 1/xn)

Where n is the number of segments, and x represents the values (speeds in this case).
1/18 + 1/21 + 1/24 + 1/17 = 0.0556 + 0.0476 + 0.0417 + 0.0588 = 0.2037
Harmonic Mean = 4 / 0.2037 = 19.64 mph (rounded)

Example (Weighted Harmonic Mean)

Weighted Harmonic Mean Calculation:
Segment #1: 21.1 km, 23 kph
Segment #2: 32.0 km, 15.6 kph
Segment #3: 29.5 km, 14.2 kph
Segment #4: 30.5 km, 10.2 kph

For weighted harmonic mean, we use:
Weighted Harmonic Mean = (w₁ * a₁ + w₂ * a₂ + w₃ * a₃ + ...) / (w₁/a₁ + w₂/a₂ + w₃/a₃ + ...)

Where w represents the distance (weights), and a represents speed/velocity (values).
(21.1 * 23 + 32.0 * 15.6 + 29.5 * 14.2 + 30.5 * 10.2) / (21.1/23 + 32.0/15.6 + 29.5/14.2 + 30.5/10.2)
= (484.3 + 499.2 + 418.9 + 311.1) / (0.915 + 2.051 + 2.078 + 2.988)
= 1713.5 / 8.032 = 14.07 kph (rounded)

Important Notes

**Use the Unweighted Harmonic Mean when velocity is the same across segments, otherwise use the Weighted Harmonic Mean when velocities (or other measures) differ across segments. This distinction ensures that each value is appropriately represented in the mean.**
