2.3 Distribution

Summary Statement of Topic

Using statistics can help you find the overall distribution of a specific topic, like demographics, using different types of distribution models.

Key Terms

  • Statistics: Branch of mathematics that involves collecting, analyzing, and displaying data
  • Distribution: A description of the possible values of a random variable and the possible occurrences of these values
  • Data: Information that has been collected to represent real life situations, usually in number form
  • Range: The difference between the smallest value and the greatest value in a data set
  • Arithmetic Mean: Also called the average; Arithmetic Mean = x‚ÇÅ + x‚ÇÇ + ... + xn
  • Standard Deviation: A measure of spread of a data set. The larger the standard deviation, the more spread out the data is.
  • Average: Also called the arithmetic mean
  • Sample: A specified part of a population, intended to represent the population as a whole
  • Population: The entire group of interest from which the sample is drawn
