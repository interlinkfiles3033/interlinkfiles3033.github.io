4.6 Calculating Variance

Big Picture

Variance helps measure how much the values in a data set vary from the mean. It calculates the average squared deviation from the mean. A high variance means the data points are spread out more widely around the mean, while a low variance means the data points are closer to the mean.

Key Terms

  • Skew: When the trend of data is skewed toward one end or the other.
  • Variance (𝝈²): The mean of the squares of the deviations of each data item from the mean. The formula is: (x₁ - x̄)² + (x₂ - x̄)² + ... + (xn - x̄)² / n
  • Data: Information that has been collected to represent real-life situations, usually in number form.
  • Deviation: A measure of the difference between a given value and the mean.
  • Outliers: An observation that lies an abnormal distance from other values in a random sample.
  • Data set: A collection of observations or values of a variable.

Variance Formula

The formula for variance is: 

(x₁ - x̄)² + (x₂ - x̄)² + ... + (xn - x̄)²
Variance (𝝈²) = ---------------------------------------------------
                                  n

Where:

  • x₁, x₂, ..., xn: Each data point in the set.
  • x̄: The mean (average) of the data set.
  • n: The number of data points.

Example of Variance Calculation

Consider the following data set: {4, 8, 6, 5, 3}

First, find the mean (x̄): 

x̄ = (4 + 8 + 6 + 5 + 3) / 5 = 26 / 5 = 5.2

Now, calculate the deviations and square them: 

(4 - 5.2)² = (-1.2)² = 1.44
(8 - 5.2)² = (2.8)² = 7.84
(6 - 5.2)² = (0.8)² = 0.64
(5 - 5.2)² = (-0.2)² = 0.04
(3 - 5.2)² = (-2.2)² = 4.84

Now sum the squared deviations: 

1.44 + 7.84 + 0.64 + 0.04 + 4.84 = 14.8

Finally, divide by the number of data points (n = 5): 

Variance = 14.8 / 5 = 2.96

Therefore, the variance of this data set is 2.96.

Important Notes

Variance is a useful measure to assess the spread of data points in a dataset. A higher variance indicates greater variability, while a lower variance indicates that the data points are closer to the mean. It is particularly useful in statistical analysis and probability theory.
