4.4 Median

Summary Statement of Topic

The median helps find the “middle number” in a data set, where exactly ½ of the data is greater, and exactly ½ is less. It provides a measure of central tendency that is not influenced by extreme values (outliers) as much as the mean.

Key Terms

  • Data Set: A collection of these observations of the variable.
  • Data: Information that has been collected to represent real-life situations, usually in number form.
  • Outlines: An observation that lies an abnormal distance from other values in a random sample from a population.
  • Extremes: The minimum and maximum value in a dataset.
  • Arithmetic Mean: Also called the average; Arithmetic Mean = x₁ + x₂ + ... + xn.
  • Histogram: A bar chart that describes a frequency distribution.
  • Interval: A specific and limited part of a function.

Example of Median Calculation

Consider the following data set: {22, 22, 33, 35, 35, 36, 39, 39, 39, 41, 42, 42, 43, 43, 44, 44, 48, 52, 54, 55, 55, 57, 57, 57}

To calculate the median, we need to order the data and then find the middle number:

Ordered data set: {22, 22, 33, 35, 35, 36, 39, 39, 39, 41, 42, 42, 43, 43, 44, 44, 48, 52, 54, 55, 55, 57, 57, 57}

Since there are 24 data points (an even number), we calculate the median as the average of the two middle numbers:
(42 + 43) / 2 = 85 / 2 = 42.5

Therefore, the median of this data set is 42.5.

Example (Odd Number of Data Points)

Now consider the data set: {22, 24, 25, 26, 27, 28, 30, 31, 35}. In this case, we have 9 values (an odd number).

To find the median, we simply locate the middle value:
{22, 24, 25, 26, 27, 28, 30, 31, 35}

The middle value is 27, so the median of this data set is 27.

Important Notes

When the data set has an even number of observations, the median is found by averaging the two middle values. This makes the median less sensitive to extreme values (outliers) compared to the arithmetic mean, which can be skewed by very high or low values.
