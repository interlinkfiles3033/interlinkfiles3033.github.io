1.1 Probability & Statistics

Key Concepts

  • Probability: Likelihood of an event occurring
  • Event: Something that can have one or more outcomes
  • Independent Events: Outcomes that don't affect each other

Probability Formula

P(A) = Number of ways A can occur / Size of Sample Space

Always between 0 (never) and 1 (always)

Dice Roll Example

Sample Space: {1, 2, 3, 4, 5, 6}

Event: Rolling a 5

Probability: 1/6
