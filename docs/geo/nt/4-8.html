4.8 Calculation Standard Deviation

Big Picture

Standard deviation is a measure that tells you how spread out the numbers in a data set are. It is calculated as the square root of the variance. The larger the standard deviation, the more spread out the data is. A smaller standard deviation means that the data points are closer to the mean.

Key Terms

  • Standard Deviation: A measure of the spread of a data set. It is the square root of the variance.
  • Variance (𝝈²): The mean of the squared deviations of each data point from the mean. The formula is: (x₁ - x̄)² + (x₂ - x̄)² + ... + (xn - x̄)² / n.
  • Sample: A specified part of a population, intended to represent the population as a whole.
  • Population: The entire group of interest from which a sample is drawn.
  • Statistics: A branch of mathematics that involves collecting, analyzing, and displaying data.
  • Deviation: A measure of the difference between a data point and the mean.
  • Units: The individuals in a population being studied.

Calculating Standard Deviation

The formula to calculate standard deviation (𝝈) is: 

𝝈 = √((x₁ - x̄)² + (x₂ - x̄)² + ... + (xn - x̄)² / n)

Where:

  • x̄: The mean (average) of the data set.
  • x₁, x₂, ..., xn: Each individual data point.
  • n: The number of data points in the data set.

Example: Calculate Standard Deviation

Consider the following data set: {4, 8, 6, 5, 3}

Step 1: Find the mean (x̄): 

x̄ = (4 + 8 + 6 + 5 + 3) / 5 = 26 / 5 = 5.2

Step 2: Calculate the squared deviations from the mean: 

(4 - 5.2)² = (-1.2)² = 1.44
(8 - 5.2)² = (2.8)² = 7.84
(6 - 5.2)² = (0.8)² = 0.64
(5 - 5.2)² = (-0.2)² = 0.04
(3 - 5.2)² = (-2.2)² = 4.84

Step 3: Sum the squared deviations: 

1.44 + 7.84 + 0.64 + 0.04 + 4.84 = 14.8

Step 4: Divide by the number of data points (n = 5): 

Variance = 14.8 / 5 = 2.96

Step 5: Take the square root of the variance to find the standard deviation: 

𝝈 = √(2.96) ≈ 1.72

Therefore, the standard deviation of the data set is approximately 1.72.

Key Takeaways

Standard deviation is a crucial statistic used to quantify the spread or variability of a data set. A larger standard deviation means the data is more spread out, while a smaller standard deviation means the data points are closer to the mean. Understanding standard deviation helps in assessing consistency and predictability within data.
