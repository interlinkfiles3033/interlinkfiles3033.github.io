4.2 Geometric Mean

Summary Statement of Topic

The geometric mean helps find the “middle” value of a set while taking into account differences in the proportion of the different ranges, making it particularly useful for data sets that contain values that are more intrinsically influential than others.

Key Terms

  • Survey: A data collection method used to gather information about individuals’ opinions, beliefs, or habits.
  • Average: The arithmetic mean is often called the average.
  • Arithmetic Mean: Also called the Average; Arithmetic Mean = x₁ + x₂ + ... + xn.
  • Geometric Mean: A method of finding the “middle” value in a set that contains some values that are intrinsically more influential than others.
  • Data: Information that has been collected to represent real life situations, usually in number form.
  • Weighting: A mathematical device used when performing a sum, integral, or average to give some elements more “weight” or influence on the result than other elements in the same set.
  • Weighted: A value or set of values takes into account varying levels of importance among members of the set.

Example

Finding interest rates over the years:

Sample: {1.07, 1.09, 1.11, 1.13, 1.15, 1.12}

Process:
N → total count of values: {1.07, 1.09, 1.11, 1.13, 1.15, 1.12} → 6
r₁ → values in the set: {1.07} → example value
Multiply all values together:
1.07 * 1.09 * 1.11 * 1.13 * 1.15 * 1.12 = 1.88
Take the 6th root of 1.88:
6√1.88 = 1.11

The geometric mean is approximately 1.11, representing an average interest rate of 11% per year.

Additional Notes

The geometric mean is especially useful when comparing different sets of data that have varying influences on the result. It is commonly used for financial data, such as interest rates and growth rates, where the data points are not equally influential.
